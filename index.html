otto

oTTo is a java web application (.war) for recording and visualising custom events.

It is used to collect information, monitor ongoing and past activity and generate reports and graphs.

10 minutes test

A maven 2 environment and a mongodb instance is required for the test.

First thing to do is to checkout source code from git

git clone git://github.com/dbourdette/otto.git

Then launch tomcat with maven

mvn tomcat:run

An otto instance should be running on http://localhost:8080

get the source code on GitHub : dbourdette/otto
